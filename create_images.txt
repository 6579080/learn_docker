для создания образа необходим dockerfile

обычно dockerfile помещают в коне папки приложения

dockerfile содержит инструкции по созданию образа
при создании образа можно указать имя и тег для образа

на оснавании образа можно создавать контейнеры

docker build . -t my-calendar

//вторая версия после внесения изменений
docker build . -t my-calendar:2.0.0

docker run -it my-calendar:2.0.0
